How it works

The recommendation is based on two different similiraty measures between articles

The topic distance

Based on the content of the abstract, we extract the topics of the articles (i.e. what is it talking about). Similar papers are then assumed to be of similar topics

The authors collaborations distance

The more authors articles share, the more similar they are.

How the final metric is done

The recommendation is based on a uniform linear combination between the two different distance measures. The detail of the recommendation for the first level neigbhors of the current paper is shown below.
